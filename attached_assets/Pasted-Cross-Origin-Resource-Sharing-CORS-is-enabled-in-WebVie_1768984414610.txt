Javascript in the Webview having CORS enabled to be loaded from file and any arbitrary URL

setAllowUniversalAccessFromFileURLs is set to True which is Insecured and found in Lcom/ digitaloperaocean/webviewcode/MainActivity;->webViewSetting (Landroid/webkit/WebView;)V
setAllowUniversalAccessFromFileURLs is set to True which is Insecured and found in Lcom/ digitaloperaocean/webviewcode/MainActivity;->webViewSetting (Landroid/webkit/WebView;)V
 
setAllowFileAccessFromFileURLs is set to True which is Insecured and found in Lcom/ digitaloperaocean/webviewcode/MainActivity;->webViewSetting (Landroid/webkit/WebView;)V



Example of an insecure code :

WebSettings settings = getSettings(); settings.setAllowUniversalAccessFromFileURLs(true) settings.setAllowFileAccessFromFileURLs(true)



Below is an example of how to prevent the application from Javascript CORS issue:

WebSettings settings = getSettings(); settings.setAllowUniversalAccessFromFileURLs(false) settings.setAllowFileAccessFromFileURLs(false)
